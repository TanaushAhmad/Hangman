# React TV Shows Hangman

This is a React-based Hangman game featuring TV show titles (2000-present).

Quick start

1. Install dependencies

```bash
npm install
```

2. Run the app

```bash
npm start
```

What’s included

- `src/components/Hangman.jsx` — main game component (adapted from `TVShowsHangman.jsx`)
- `src/App.js`, `src/index.js` — React app entry
- `public/index.html`
- `package.json` — scripts and dependencies
